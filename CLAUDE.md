## About this repo

This is a personal knowledge bank (`yj-knowledge-bank`). It stores reference guides and deep-dive technical documentation. There is no code, no build system, and no tests.

## Content structure

Each markdown file is a self-contained deep-dive guide on a single technical topic, written in Chinese. The guides are designed to be comprehensive reference material — they explain the full mental model behind a technology, from foundational concepts to implementation details.

## Writing style for new guides

- Written in Chinese (Simplified)
- Bottom-up pedagogy: start from fundamental concepts before layering on specifics
- Dense and thorough — prefer diagrams, tables, and step-by-step tracing over brief summaries
- Include real configuration snippets where applicable
- Cross-reference related concepts with section pointers
- Use ASCII art diagrams for architecture overviews and protocol flows
- End with a "关键概念速查表" summary table for quick lookup
- Cite RFCs and authoritative sources where relevant
