## 核心概念速览

在深入阅读之前，这里有一个最简概念地图，帮助建立心智模型：

| 概念 | 一句话解释 |
|------|-----------|
| 大模型 (LLM) | 用海量数据训练的、参数规模巨大的语言模型 |
| Transformer | 现代大模型的基础架构，核心是 Self-Attention 机制 |
| Token | 模型处理文本的最小单位（不是字，不是词，介于之间） |
| 预训练 (Pretrain) | 在海量语料上学习语言的基础知识 |
| 微调 (Fine-tune) | 在特定任务数据上调整模型行为 |
| 涌现能力 | 模型大到一定程度后「突然」获得的能力 |
| 量化 (Quantization) | 降低模型精度以减小体积和加速推理 |

## 关键概念速查表

| 中文概念 | 英文 | 对应笔记 | 一句话解释 |
|----------|------|----------|-----------|
| 大语言模型 | LLM (Large Language Model) | [[什么是大模型]] | 参数量达数十亿以上的语言模型，展现出强大的语言理解和生成能力 |
| 缩放法则 | Scaling Law | [[什么是大模型]] | 模型性能随参数量、数据量、计算量的增加而可预测地提升 |
| 涌现 | Emergence | [[什么是大模型]] | 模型规模突破某个阈值后，突然展现出训练时未明确教授的能力 |
| 词元 | Token | [[什么是大模型]] | 文本被分割后的最小处理单元，是模型输入的基本单位 |
| Transformer | Transformer | [[大模型技术原理]] | 基于自注意力机制的神经网络架构，是现代大模型的基石 |
| 自注意力 | Self-Attention | [[大模型技术原理]] | 让序列中每个位置都能关注到所有其他位置的机制 |
| 多头注意力 | Multi-Head Attention | [[大模型技术原理]] | 并行运行多组注意力，捕捉不同类型的语义关系 |
| 位置编码 | Positional Encoding | [[大模型技术原理]] | 为 Transformer 注入序列顺序信息的方法 |
| 预训练 | Pretraining | [[大模型训练流程]] | 在海量无标注语料上训练模型学习语言基础知识 |
| 监督微调 | SFT (Supervised Fine-Tuning) | [[大模型训练流程]] | 使用高质量问答对调整模型，使其学会对话格式 |
| 人类反馈强化学习 | RLHF | [[大模型训练流程]] | 利用人类偏好反馈训练奖励模型，再用强化学习优化模型 |
| 直接偏好优化 | DPO | [[大模型训练流程]] | 直接使用偏好数据优化模型，无需显式训练奖励模型 |
| 幻觉 | Hallucination | [[大模型推理与优化]] | 模型生成的内容看似合理但与事实不符 |
| KV 缓存 | KV Cache | [[大模型推理与优化]] | 缓存已计算的 Key 和 Value，避免自回归生成时的重复计算 |
| 量化 | Quantization | [[大模型推理与优化]] | 将模型参数从高精度（FP16）压缩到低精度（INT4/INT8） |
| 视觉语言模型 | VLM (Vision-Language Model) | [[模型分类与对比]] | 能同时理解图像和文本的多模态模型 |
| 扩散模型 | Diffusion Model | [[模型分类与对比]] | 通过逐步去噪生成图像的模型，是图像生成的主流架构 |
